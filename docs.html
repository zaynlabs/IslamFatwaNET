🚀 Schnellstart & Installation

Willkommen beim Islam Fatwa Discord Bot! Mit diesem Bot holst du dir das gesamte Wissen und die Urteile von islamfatwa.net direkt auf deinen Discord-Server. Keine Panik, die Installation ist kinderleicht.

🌟 Der einfachste Weg (für Windows)

Wenn du Windows benutzt, haben wir einen praktischen Einrichtungs-Assistenten für dich vorbereitet. Du musst keine Befehle in ein Terminal eingeben!

  1. Lade die Bot-Dateien auf deinen Computer herunter und entpacke sie.
  2. Doppelklicke auf die Datei starten.bat im Ordner.
  3. Der Assistent öffnet sich und fragt dich nach deinem Bot-Token. Gib ihn einfach ein und drücke Enter – fertig!

💡 Was macht der Einrichtungs-Assistent?

Er installiert automatisch alle notwendigen Programmpakete (Bibliotheken), erstellt deine Konfigurationsdatei (.env) und fragt dich, ob du den Bot direkt starten willst.

💻 Manueller Start (für erfahrene Nutzer)

Wenn du lieber die Konsole nutzt oder ein anderes Betriebssystem hast, führe einfach folgende Befehle aus:

# 1. Abhängigkeiten installieren
pip install -r requirements.txt

# 2. Bot starten
python main.py

🤖 Verfügbare Befehle

Der Bot benutzt ausschließlich moderne Slash-Commands. Tippe einfach einen Schrägstrich (/) in deinen Discord-Chat, um die Befehle zu sehen.

{
  "/suche": {
    "beschreibung": "Öffnet das interaktive Suchmenü direkt in Discord.",
    "optionen": [
      "suchbegriff (Was suchst du?)",
      "kategorie (Filter nach Themen)",
      "gelehrter (Filter nach Gelehrten)"
    ]
  },
  "/daily": {
    "beschreibung": "Postet das aktuelle Fatwa des Tages manuell in den Chat."
  },
  "/random": {
    "beschreibung": "Postet ein völlig zufälliges Rechtsurteil aus der Datenbank."
  },
  "/einstellungen": {
    "beschreibung": "Öffnet das Einstellungs-Panel für Server-Admins."
  }
}

🔍 Wie funktioniert die Suche?

Wenn du /suche eingibst, öffnet sich ein schickes Such-Dashboard. Dort kannst du über zwei Auswahllisten (Kategorie und Gelehrter) filtern oder über den Button Suchwort ändern einen Begriff eintippen. Der Bot durchsucht daraufhin blitzschnell die Website und zeigt dir die besten Ergebnisse.

⚙️ Einstellungen & Design

Du kannst das Design des Bots (Farben, Knöpfe, Vorschaufenster) ganz einfach anpassen. Dafür gibt es zwei Wege.

🛠️ Weg A: In Discord anpassen (Empfohlen)

Tippe einfach /einstellungen im Discord-Chat ein (du musst Server-Administrator sein):

  • Theme-Farbe: Ändere die Farbe der Infokarten (Embeds).
  • Kanal festlegen: Setze den aktuellen Kanal für das automatische tägliche Posten.
  • Vorschau: Schalte die Vorschaubilder (Thumbnails) an oder aus.
  • Speichere danach einfach mit dem Speichern-Knopf ab.

📄 Weg B: In settings.json anpassen

Du kannst die Datei settings.json im Ordner des Bots auch direkt mit einem Editor bearbeiten:

{
    "embed_color": "0x0a58ca", // Theme-Farbe der Embeds
    "search_limit": 3,          // Max. Suchtreffer parallel laden
    "bookmark_button_style": "success", // Button-Farbe
    "daily_fatwa_enabled": true, // Täglicher automatischer Post
    "daily_fatwa_mode": "latest",  // "latest" (Neuestes) oder "random" (Zufall)
    "daily_channel_id": 1234567890 // Discord Kanal-ID
}

🔖 Das Lesezeichen-System

Unter jedem geposteten Fatwa findest du einen Button namens Als Lesezeichen speichern. Wenn ein User darauf klickt, passiert Folgendes:

  1. Der Bot schickt dem User das komplette Urteil (inklusive Frage und Antwort) als private Direktnachricht (DM).
  2. So hat der User sein persönliches Lesezeichen dauerhaft in seinem Postfach und kann es jederzeit nachlesen.

⚠️ Wichtiger Hinweis für deine Server-Mitglieder!

Damit der Bot Lesezeichen schicken kann, müssen User Direktnachrichten auf dem Server erlauben. Wenn der Bot eine Fehlermeldung schickt, hilft ein Rechtsklick auf das Server-Icon > Privatsphäre-Einstellungen > "Direktnachrichten von Servermitgliedern erlauben" aktivieren.

⚠️ Fehlersuche (Troubleshooting)

Hier sind Lösungen für die häufigsten Probleme, falls mal etwas nicht klappt.

❌ Der Bot startet nicht / wirft Fehler

  • Problem: Du hast deinen Token nicht richtig eingetragen.
  • Lösung: Prüfe die Datei .env im Hauptordner. Dort muss BOT_TOKEN=deintokenhier stehen. Achte darauf, dass keine Leerzeichen oder Anführungszeichen um den Token stehen.

❌ Slash-Commands tauchen in Discord nicht auf

  • Problem: Discord braucht manchmal ein paar Minuten, um neue Befehle anzuzeigen.
  • Lösung: Warte ca. 5 Minuten oder starte deinen Discord-Client (App) einmal komplett neu. Stelle sicher, dass der Bot in der Konsole meldet: "Befehl(e) global synchronisiert!".

❌ Lesezeichen-Speichern schlägt fehl

  • Problem: Der Empfänger hat seine DMs geschlossen.
  • Lösung: Der User muss DMs für Mitglieder dieses Servers erlauben (siehe Beschreibung unter "Lesezeichen").

❌ Der tägliche Fatwa-Post kommt nicht

  • Problem: Entweder ist der tägliche Post in `/einstellungen` deaktiviert, die Kanal-ID ist falsch oder der Bot hat keine Berechtigung, in diesem Kanal Nachrichten zu schreiben.
  • Lösung: Prüfe in Discord, ob der Bot die Rolle "Nachrichten senden" und "Einbettungen hinzufügen" in dem eingestellten Kanal besitzt.
