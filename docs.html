Schnellstart & Einrichtung

Willkommen beim Islam Fatwa Bot! Mit diesem Bot kannst du Rechtsurteile von islamfatwa.net direkt auf deinen Discord-Server holen. Die Einrichtung ist extrem einfach und schnell erledigt.

✨ Automatische Einrichtung (Windows)

Nutzt du Windows? Dann kannst du den vollautomatischen Einrichtungs-Assistenten nutzen:

  1. Lade die Bot-Dateien auf deinen Computer herunter und entpacke sie.
  2. Doppelklicke auf die Datei starten.bat im Hauptordner.
  3. Der Assistent öffnet sich und führt dich komplett durch das Eintragen des Bot-Tokens. Fertig!
💡

Was macht starten.bat?

Die Datei prüft, ob die Programmpakete installiert sind (installiert sie falls nötig), erstellt die Konfigurationsdatei und startet deinen Bot ganz unkompliziert per Doppelklick.

💻 Manueller Start

Für erfahrene Nutzer oder andere Betriebssysteme lässt sich der Bot auch manuell über die Konsole starten:

pip install -r requirements.txt
python main.py

Verfügbare Bot-Befehle

Der Bot verwendet ausschließlich moderne Slash-Commands. Gib einfach einen Schrägstrich (/) im Discord-Chat ein, um alle Befehle anzuzeigen.

🔍 /suche
Öffnet das interaktive Such-Dashboard. Filtere nach Suchbegriff, Kategorie oder Gelehrten.
🕌 /daily
Postet das aktuelle Fatwa des Tages manuell in den Chatkanal.
🔄 /random
Postet ein zufälliges Rechtsurteil aus der islamfatwa.net Datenbank.
⚙️ /einstellungen
Öffnet das Admin-Einstellungsmenü zur Anpassung von Farbe, Kanal und Vorschau.

💡 Interaktive Suche nutzen

Nach Eingabe von /suche zeigt der Bot dir ein Dashboard mit Filtermöglichkeiten. Du kannst bequem Kategorien oder Gelehrte über Dropdown-Menüs auswählen oder über den Stift-Button einen eigenen Suchbegriff eintippen.

Bot-Einstellungen anpassen

Du kannst das optische Erscheinungsbild des Bots sowie das automatische Posten direkt in Discord anpassen.

🎨 /einstellungen im Chat

Wenn du Administrator auf dem Discord-Server bist, kannst du das Konfigurations-Panel aufrufen:

  • Theme-Farbe: Wähle eine Designfarbe für die Botschaften (z. B. Königsblau oder Smaragdgrün).
  • Posting-Kanal: Lege den Kanal fest, in dem das tägliche Fatwa gepostet wird.
  • Bilder-Vorschau: Schalte Vorschaubilder (Thumbnails) an oder aus.

⚙️ Manuell in settings.json

Die Einstellungen werden in der Datei settings.json gespeichert und können auch direkt editiert werden:

{
    "embed_color": "0x0a58ca",
    "search_limit": 3,
    "bookmark_button_style": "success",
    "daily_fatwa_enabled": true,
    "daily_fatwa_mode": "latest",
    "daily_channel_id": 123456789
}

Das Lesezeichen-System

Unter den Suchtreffern und Posts befindet sich ein Knopf mit der Aufschrift Als Lesezeichen speichern.

🌟 So funktioniert es

Klickt ein User auf diesen Knopf, sendet der Bot ihm die komplette Nachricht inklusive Frage und Antwort als private Direktnachricht (DM). So kann sich jeder User eine eigene Bibliothek nützlicher Rechtsurteile in seinen DMs anlegen.

⚠️

Häufige Fehlerquelle

Wenn User die Fehlermeldung erhalten, dass Nachrichten nicht zugestellt werden können, haben sie ihre Privatsphäre-Einstellungen gesperrt. Sie müssen DMs von Servermitgliedern in den Discord-Einstellungen erlauben.

Häufige Fragen & Fehlersuche

Falls einmal etwas nicht rund laufen sollte, findest du hier schnelle Lösungen.

⚠️ Der Bot startet nicht

Vergewissere dich, dass du deinen Bot-Token korrekt in die Datei .env eingetragen hast. Das Format sollte so aussehen:

BOT_TOKEN=DeinDiscordBotTokenOhneAnfuehrungszeichen

⚠️ Slash-Befehle werden nicht angezeigt

Wenn die Slash-Befehle (wie /suche) nicht in deinem Chat erscheinen, starte dein Discord-Programm neu. Der Bot registriert die Befehle global bei Discord – das kann nach dem ersten Start manchmal bis zu 5 Minuten douern.

⚠️ Bot schreibt keine Nachrichten

Vergewissere dich, dass der Bot in den Kanaleinstellungen die Berechtigung besitzt, Nachrichten zu senden und Einbettungen (Links) hinzuzufügen.
